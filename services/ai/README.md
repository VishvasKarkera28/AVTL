# AI Services

Target stack: Python with TensorFlow, PyTorch, or scikit-learn depending on the model.

## Services

- Driver risk scoring.
- Predictive maintenance.
- Theft anomaly detection.
- Damage detection.
- EV energy optimization.

## Production Requirements

- Store model version for every prediction.
- Store explanation factors for human review.
- Never hide raw evidence behind only a score.
- Support batch and realtime inference.
- Keep training data separated by privacy and tenant boundaries.
- Create model monitoring for drift, false positives, and data gaps.
